# Content 폴더 구조

연구 관련 콘텐츠를 체계적으로 관리합니다.

## 폴더 구조

```
content/
├── data/          # 데이터 파일 (CSV, Excel, Pickle 등)
├── figures/       # 생성된 그래프 및 시각화
├── tables/        # 결과 테이블
└── code/          # 재사용 가능한 코드 스니펫
```

## 사용 가이드

- **data/**: 원본 데이터 및 전처리된 데이터 저장
- **figures/**: Quarto에서 생성된 그래프 자동 저장
- **tables/**: 분석 결과 테이블 저장
- **code/**: 공통 함수 및 유틸리티 코드
