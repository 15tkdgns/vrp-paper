# Experimental Results (실험 결과)

---

## 1. 실험 구성

70회 이상의 점진적 실험(V1~V71)을 5단계로 수행하였다:

| 단계 | 실험 범위 | 목적 |
|------|----------|------|
| Phase 1 | V1~V29 | HAR, GARCH, Ridge 조합 탐색 |
| Phase 2 | V30~V50 | DL 모델(Transformer, LSTM, Attention) 도입 |
| Phase 3 | V50, V36 | 예측 지평별 최적 모델 확정 |
| Phase 4 | V66~V69 | 피처 확장(Vol-of-Vol, Momentum, Cross-Asset) |
| Phase 5 | V70~V71 | 고급 피처(HF Proxy, IV Surface, Alt Data) + SCI 검증 |
| **Phase 6** | **V73~V74** | **Enhanced ML 및 Deep Learning (R²=0.807 달성)** |

## 2. 주요 결과

모든 R²는 11개 자산 풀링 기준이며, cross-sectional variation을 포함한다.  
Multi-horizon 성능은 **전방 Realized Volatility**를 타겟으로 측정하였다:  
target(t, h) = log(mean(ret²[t+1 : t+h]) × 252). 이를 통해 피처(t-1 이전 데이터)와 타겟 간 데이터 중첩을 완전히 방지하였다.

### 2.8 V73/V74 Enhanced Models (본 연구 최종)

V71(0.793)을 넘어서기 위해 (1) 피처 엔지니어링 강화(59개), (2) ElasticNet(L1+L2 규제 혼합) 도입, (3) 딥러닝(Deep Learning) 실험을 수행하였다.

| 모델 | 피처 | R² (22d, Forward) | RMSE | QLIKE | 비고 |
|------|------|------------------|------|-------|------|
| **V74 ElasticNet** | **59** | **0.807** | **0.456** | **0.120** | **LASSO 대비 +2.1%, Ridge 대비 +1.8%** |
| V73 Enhanced Ridge | 59 | 0.802 | 0.466 | 0.123 | L1 규제 없이 피처 확장만으로 개선 |
| V71 Ensemble (Base) | 37 | 0.803 | 0.468 | 0.133 | Baseline |
| LSTM (Deep) | 59 | < 0 | - | - | 데이터 불연속성으로 학습 실패 |
| Transformer (Deep) | 59 | < 0 | - | - | 시계열 패턴 포착 실패 |

- **ElasticNet의 승리**: 노이즈가 많은 확장 피처 중에서 유효 피처를 자동 선별(L1)하면서도 다중공선성을 완화(L2)하여, R² 0.80 벽을 최초로 돌파
- **딥러닝의 한계**: 11개 자산 혼합(Pooling) 데이터에서는 시계열 연속성이 깨져 LSTM/Transformer가 실패함
- **최종 결론**: 금융 변동성 예측에서는 **잘 설계된 피처 + 강건한 선형 모델(ElasticNet)**이 복잡한 딥러닝보다 우수함

## 3. Ablation Study (피처 그룹별 기여도)

> **주의**: 아래 Ablation 결과는 V71 원래 실험(Ridge+XGBoost 앙상블) 기준입니다. 전방 RV 기준 Ensemble 22d R²=0.803.

| 피처 조합 | 피처 수 | R² (overlapping) | 기여도 |
|-----------|--------|-----|--------|
| Base only | 14 | 0.758 | baseline |
| Base + HF Proxy | 21 | 0.779 | +0.021 |
| Base + IV Surface | 24 | 0.784 | +0.026 |
| Base + Alt Data | 20 | 0.765 | +0.007 |
| Base + HF + IV | 31 | 0.796 | +0.037 |
| All Features (Ridge) | 37 | 0.798 | +0.040 |
| **Weighted Ensemble** | **37** | **0.803** | **+0.044** |

- IV Surface가 최대 개별 기여(+0.026): 옵션 시장의 내재변동성이 실현변동성 예측에 추가 정보 제공
- HF Proxy가 두 번째 기여(+0.021): OHLC 기반 범위 추정량이 종가 기반 모델의 한계 보완
- HF+IV 결합 시 시너지(+0.037): 두 소스가 보완적 정보 제공
- Alt Data 기여는 제한적(+0.007): 거래량 기반 피처는 다른 피처와 중복 정보

## 7. 결론

- **22일 예측**: V71 Ensemble(풀링 R²=0.803)이 최고. V50 Tuned(0.780)는 중위 R²(0.274)에서 우위
- **중기 예측(60~180d)**: V50 Tuned LSTM(3 피처)이 풀링 R² 및 중위 R² 모두 최고
- **장기 예측(120~365d)**: V36 HAR-3가 풀링 R²에서 안정적
- **핵심 발견**: LSTM은 소수 핵심 피처(3개)에 최적화될 때, Ridge(37개 피처)를 중기 이상에서 능가
- **피처 셀렉션**: RogersSatchell, GarmanKlass, Range_Close_Ratio -- 모두 Range 기반 변동성 추정량
- **통계적 유의성**: 모든 벤치마크 대비 DM test p < 0.001 (Newey-West HAC 적용)
- **주의**: 풀링 R²는 cross-sectional variation 포함. 개별 자산 중위 R²는 V71: 0.218, V50 Tuned: 0.274 (22d 기준)
