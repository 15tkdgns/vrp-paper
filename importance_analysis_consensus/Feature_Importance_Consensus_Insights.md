# Feature Importance Analysis Consensus: Key Insights

새로 수행하신 **"Feature Importance Analysis Consensus"** 추가 실험 결과를 분석하여 도출한 핵심 인사이트입니다.

## 1. 모델 간 변수 중요도 평가의 높은 일치도 (Kendall's W)
* **결과**: 모든 예측 기간(1d ~ 252d)에 대해 p-value가 0.000으로 나타나, 선형 모델(Ridge)과 트리 기반 모델(XGBoost, Random Forest) 등 서로 다른 구조의 모델들이 중요하게 생각하는 변수의 순위가 통계적으로 유의미하게 일치(Consensus)했습니다.
* **인사이트**: 
  * особенно 5일(0.577)과 22일(0.588) 예측 기간에서 일치도가 가장 높게 나타납니다. 이는 중기 예측(1주~1달)에서 우수한 설명력을 지니는 핵심 변수들이 모델에 관계없이 공통적으로 작용하고 있음을 시사합니다.

## 2. 기간(Horizon)별 핵심 변수군의 변화 동학
Bootstrap 신뢰구간(CI)과 랭킹 결과를 분석한 결과, 예측하려는 기간의 길이에 따라 핵심 변수의 성격이 변화하는 흥미로운 패턴이 발견되었습니다.

* **초단기 (1d)**: 
  * `Parkinson_5`, `Garch_Daily`와 같이 **짧은 윈도우(5일, 일간)를 가진 고주파(HF) 및 GARCH 변수들**이 최상위권에 포진했습니다. 하루 앞을 예측하기 위해서는 최근 며칠간의 미시적 변동성 동인이 가장 중요하게 작용합니다.
* **중기 (22d)**: 
  * `RogersSatchell_22`, `GarmanKlass_22`와 같이 **22일 윈도우를 가진 고주파 특성**들이 1~2위로 올라옵니다. 또한 `Garch_Weekly`의 영향력이 강해집니다. 예측 기간과 특성을 계산하는 윈도우 기간이 일치(matching)할 때 예측력이 극대화되는 경향성이 뚜렷합니다.
* **장기 (60d 이상)**: 
  * `Garch_Weekly`, `RogersSatchell_22`, `GarmanKlass_22` 등 장기적인 추세를 반영하는 변동성 특성들이 흔들림 없이 최상단(Rank 1~5)을 차지합니다. 또한 `IV_VRP_ma22`(내재변동성 프리미엄 누적)와 같은 거시적 내재 정보의 중요도가 상승합니다.

## 3. 고주파 변수(HF Proxy)의 압도적인 우위와 견고성 (Robustness)
* **결과**: 거시 기간에 상관없이 Top 5 Consensus Rank에 늘 `RogersSatchell_22`, `GarmanKlass_22`, `Parkinson_5/22`와 같은 **HF Proxy** 변수군이 절반 이상을 차지했습니다.
* **인사이트**: 
  * 딕셔너리 형태의 단순 종가 수익률보다 일중(intraday) 고가/저가/시가를 포괄하는 HF Proxy가 미래 수익률 변동성(VRP) 예측에 훨씬 더 풍부하고 강건한 정보를 제공한다는 강력한 증거입니다. Bootstrap 신뢰 구간 또한 매우 타이트(예: CI Lower 2.1 ~ CI Upper 4.9)하게 나타나 이 변수들의 상대적 우위성에 매우 높은 확신(Confidence)을 가질 수 있습니다.

## 4. 특성 그룹별 기여도 (Ablation Study)
* **결과** (Weighted Ensemble 기준 최고 성능 0.8020 달성):
  * **IV Surface 변수군 기여도**: +0.0259 (가장 큰 개별 성능 향상)
  * **HF Proxy 변수군 기여도**: +0.0206
  * **Alternative Data 변수군 기여도**: +0.0067
* **인사이트**:
  * 순위(Rank) 관점에서는 HF Proxy가 지배적이지만, 기존 기본 변수(Base: HAR, GARCH)에 추가되었을 때 **전체 예측력(R²)을 가장 크게 끌어올리는 "비직교적/상호보완적" 정보는 내재변동성 곡면(IV Surface)**에서 나옵니다.
  * 즉, 과거 실현 변동성/고주파 정보(HF/Base)가 잡아내지 못하는 시장 참여자들의 **'미래 전망 프리미엄(IV)'**이 더해질 때 모델의 전체 퍼포먼스가 비약적으로 상승합니다. 반면 Alternative 변수들은 설명력 기여가 미미했습니다.
