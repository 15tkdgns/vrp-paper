부록 D. 데이터 전처리 및 학습 알고리즘 의사코드
알고리즘 D-1. 데이터 전처리 및 피처 생성
입력: OHLCV 일별 데이터 (11 ETFs, 2010-2024), VIX/VIX3M/VIX9D
출력: 피처 행렬 X, 타깃 벡터 y (자산 × 날짜 패널)

1. 각 자산별 결측치 처리 (forward fill)

2. 수정 종가 기반 로그수익률 r_t = log(P_t / Pt-1) 계산

3. HAR 기반 피처 계산:

- LogRV_lag{1,5,10,22}, LogRV_Std{5,22}, RV_Mom{5,22}
- Ret_lag1, Ret_abs_lag1, SPY_LogRV, Corr_SPY

4. HF Proxy 피처 계산 (원시 OHLC 사용):

- Parkinson5,22, GarmanKlass_22, RogersSatchell_22
- Range_Close_Ratio, Overnight_Vol, Overnight_Ret

5. IV Surface 피처 계산:

- IV_VIX, IV_VIX_chg, IV_VIX_ma5, IV_VIX_std5
- IV_VIX3M, IV_VIX_TermSlope, IV_VIX9D, IV_VIX_ShortSlope
- IV_VRP (= (VIX/100)2 - RV_SPY_22d), IV_VRP_ma22

6. Alternative 피처 계산:

- AltVol_Amihud, AltVol_Vol_Ratio, AltVol_PV_Corr
- AltVol_Vol_Surprise, AltVol_Order_Imbalance, AltVol_Kyle_Lambda

7. 전방 RV 타깃 yi,t(h) = 연율화(mean(r2t+1, ..., r2t+h)) 계산

8. 표본 끝부분 h일 제거

9. 피처 행렬 X, 타깃 y 반환

알고리즘 D-2. WEns 학습 및 예측 절차
입력: 피처 행렬 X, 타깃 y, 예측 기간 h, 자산군 그룹 G
출력: 테스트 구간 예측값 y

1. 전체 표본을 날짜순 정렬 후 80/20 분할

- train_end = 전체 날짜의 80% 시점
- purge gap 적용: test_start = train_end + h

2. 자산군별 독립 학습 (g ∈ G):

a. X_train_g, y_train_g 추출 (train_end까지)
b. 내부 검증셋(inner holdout) 분할 (train_g의 80/20, purge gap h 적용)
c. Ridge 하이퍼파라미터 탐색 (α grid, 내부 검증셋 R2 기준)
d. XGBoost 하이퍼파라미터 탐색 (grid search, 내부 검증셋 R2 기준)
e. WEns 가중치 탐색: pw ∈ {0.4,...,0.9}에서 내부 검증셋 R2 최대화
f. 최적 파라미터로 전체 train_g에 Ridge, XGBoost 재학습

3. 테스트 구간 예측:

- y_hat_Ridge = Ridge.predict(X_test_g)
- y_hat_XGB = XGBoost.predict(X_test_g)
- y_hat_WEns = pw * y_hat_Ridge + (1-pw) * y_hat_XGB

4. Pooled R2, Median R2, RMSE 계산

5. DM 검정 (Newey-West HAC, bandwidth=h) 수행

6. y_hat_WEns 반환
