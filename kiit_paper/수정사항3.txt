파일 기준으로 **“어디(위치)에 어떤 메모가 달렸는지”를 구조적으로 정리**하면 아래처럼 볼 수 있어. (실제 리뷰 대응용으로 바로 쓸 수 있게 위치–내용–의미까지 연결했어)

---

# 📌 1. 논문 전반(제목/초록/용어)

### 📍 위치: 제목 + Abstract + 전체 본문

**메모 내용**

* 제목 용어 불일치 (Weighted Ensemble vs 가중 앙상블)
* Cross-Asset 용어 거의 사용 안 됨 → 제목 수정 필요
* 영문/한글 혼용 문제
* 약어 full-form 일부 없음


👉 의미
→ **논문 전체에서 용어 체계가 정리 안 된 상태**
→ “형식 완성도 부족”으로 판단됨

---

# 📌 2. 서론(Ⅰ. 서론)

### 📍 위치: 서론 중간 (VIX 그림 부분)

**메모 내용**

* “그림 1 설명 없음”

👉 의미
→ 그림이 논문 논리 흐름에 연결 안 됨

---

### 📍 위치: 서론 후반 (연구 기여 나열 부분)

**메모 내용**

* “문장이 완성되지 않은 것으로 보임”
* “첫째가 누락됨”


👉 의미
→ **기여(contribution) 구조 자체가 깨짐**
→ reviewer 입장에서 가장 치명적인 부분

---

# 📌 3. 관련연구(Ⅱ)

### 📍 위치: 관련연구 전체

**메모 내용**

* 문헌 나열 수준
* 장단점 분석 부족
* 제안 연구 필요성 연결 부족
* 두 흐름(GARCH vs ML) 구분 불명확


👉 의미
→ **논문의 “왜 이 연구가 필요한가”가 설득 안 됨**

---

# 📌 4. 데이터/방법론(Ⅲ)

### 📍 위치: 3.1 자산 구성

**메모 내용**

* 일부 설명 삭제 가능

👉 의미
→ 불필요한 서술 존재 (밀도 낮음)

---

### 📍 위치: 3.3 데이터 정의

**메모 내용**

* RV, VRP 등 수식으로 표현 필요
* 수식 깨짐 문제 있음


👉 의미
→ **수식 기반 논문으로서 형식 미흡**

---

### 📍 위치: 수식 전반

**메모 내용**

* 모든 수식 → 수식 편집기로 재작성 필요


👉 의미
→ 학회 제출 형식 기준 미충족

---

# 📌 5. 모델/수식 부분

### 📍 위치: WEns 수식

**메모 내용**

* 수식 편집기로 작성 + 캡션 번호 추가 필요


👉 의미
→ 핵심 모델 정의가 형식적으로 미완성

---

# 📌 6. 결과(IV)

### 📍 위치: 그림 2 (히트맵)

**메모 내용**

* 숫자 크기 작음 → 가독성 개선 필요


---

### 📍 위치: 표 8~12

**메모 내용**

* 첫 행 bold 처리 필요


---

# 📌 7. 전체 문서 표현 문제

### 📍 위치: 논문 전반

**메모 내용**

* 띄어쓰기 오류
* 용어 불일치 (테스트세트 vs 테스트셋)
* 붙어쓰기 오류


👉 의미
→ **기초적인 문장 완성도 문제**

---

# 📌 8. 부록 / 참고문헌

### 📍 위치: 부록

**메모 내용**

* 학회 규정상 부록 불가 → 본문 이동 필요

---

### 📍 위치: 참고문헌

**메모 내용**

* DOI 추가 필요


---

# 📌 전체 구조 요약 (핵심만 압축)

| 위치    | 문제 유형 | 핵심 문제      |
| ----- | ----- | ---------- |
| 제목/초록 | 용어    | 용어 불일치     |
| 서론    | 논리    | 기여 구조 깨짐   |
| 관련연구  | 논리    | 필요성/차별성 부족 |
| 방법론   | 형식    | 수식 표현 부족   |
| 모델    | 형식    | 수식 정리 부족   |
| 결과    | 가독성   | 표/그림 미완성   |
| 전체    | 문장    | 띄어쓰기/표현 문제 |
| 부록    | 규정    | 학회 규정 위반   |

# 📌 한 줄 핵심 정리

👉 메모는 크게 3축으로 나뉨:

**1) 논리 부족 (관련연구·기여)**
**2) 형식 미흡 (수식·표·그림)**
**3) 기본 완성도 (용어·문장·규정)**
