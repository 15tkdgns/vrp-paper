예측 지평(Horizon)에 따른 최적 모델

변동성 예측에서는 예측 지평(horizon)에 따라 최적 모델이 달라집니다. 본 연구에서는 1일(단기)과 22일(장기) 예측 지평 각각에서 최고 성능을 보이는 모델의 구조를 상세히 분석했습니다.

핵심 발견
  • 1일 예측: 복잡한 비선형 패턴을 학습하는 딥러닝(V50)이 유리
  • 22일 예측: 자산별 특성을 안정적으로 반영하는 적응형 선형모델(V36)이 유리

1. 예측 지평별 성능 비교

2. 1일 예측 Champion: V50 (Dual Attention LSTM)

단기 예측에서는 시계열의 복잡한 비선형 의존성을 학습하는 능력이 중요합니다.

모델 구조 (V50)

V50은 양방향 LSTM(Bi-LSTM)과 이중 어텐션(Dual Attention) 메커니즘을 결합한 하이브리드 구조입니다.

  • R² Score: 0.821 (vs V29 Baseline 0.730, +12.5% 개선)
  • Diebold-Mariano Test: p-value < 0.001 (통계적으로 유의미한 개선)
  • Walk-Forward Validation: OOS R² = 0.820 (과적합 없음 확인)

3. 22일 예측 Champion: V36 (Asset-Class Adaptive Ridge)

장기 예측에서는 모델이 너무 유연하면(Overfitting) 노이즈까지 학습합니다. V36은 자산군별로 최적화된 단순함을 유지하여 최고 성능을 달성했습니다.

  • R² Score: 0.755 (vs V29 Baseline 0.641, +17.8% 개선)
  • 채권 카테고리: R² = 0.95 (금리 정책 추세가 예측 가능성 높임)
  • 주식 카테고리: R² = 0.51 (시장 효율성으로 랜덤워크에 가까움)

4. 왜 예측 지평에 따라 최적 모델이 다른가?

핵심 인사이트
  • 1일 예측: 시그널이 강하고 자기상관성이 높아 복잡한 딥러닝이 미세한 패턴까지 포착 가능
  • 22일 예측: 노이즈가 많고 구조적 변화(Structural Break) 영향이 커서 단순하고 강건한 모델이 유리

장기 리스크 관리 (22일 예측) V36 (Asset-Adaptive) 0.755 자산군별 특성 반영에 탁월

## 예측 지평(Horizon)에 따른 최적 모델

변동성 예측에서는 **예측 지평(horizon)**에 따라 최적 모델이 달라집니다. 본 연구에서는 1일(단기)과 22일(장기) 예측 지평 각각에서 최고 성능을 보이는 모델의 구조를 상세히 분석했습니다.

::: {.callout-note}
## 핵심 발견
- **1일 예측**: 복잡한 비선형 패턴을 학습하는 **딥러닝(V50)**이 유리
- **22일 예측**: 자산별 특성을 안정적으로 반영하는 **적응형 선형모델(V36)**이 유리
:::

## 1. 예측 지평별 성능 비교

```{python}
#| echo: false
#| fig-width: 10

import matplotlib.pyplot as plt
import numpy as np

# 한글 폰트 설정
plt.rcParams['font.family'] = 'UnDotum'
plt.rcParams['axes.unicode_minus'] = False

models = ['V29\n(베이스라인)', 'V36\n(자산 적응형)', 'V43\n(트랜스포머)', 'V50\n(이중 어텐션 LSTM)', 'V42\n(앙상블)']
r2_1day = [0.730, 0.755, 0.797, 0.821, 0.810]  # 1-day horizon
r2_22day = [0.641, 0.755, 0.534, 0.503, 0.656]  # 22-day horizon

x = np.arange(len(models))
width = 0.35

fig, ax = plt.subplots(figsize=(10, 5))
bars1 = ax.bar(x - width/2, r2_1day, width, label='1일 예측 지평', color='#4CAF50', alpha=0.8)
bars2 = ax.bar(x + width/2, r2_22day, width, label='22일 예측 지평', color='#2196F3', alpha=0.8)

ax.set_ylabel('R² 스코어', fontsize=12)
ax.set_title('예측 지평별 성능 비교', fontsize=14, fontweight='bold')
ax.set_xticks(x)
ax.set_xticklabels(models, fontsize=10)
```

## 2. 1일 예측 Champion: V50 (Dual Attention LSTM)

## 3. 22일 예측 Champion: V36 (Asset-Class Adaptive Ridge)

장기 예측에서는 모델이 너무 유연하면(Overfitting) 노이즈까지 학습합니다. V36은 자산군별로 최적화된 단순함을 유지하여 최고 성능을 달성했습니다.

### 모델 구조 (V36)

V36은 **HAR-GARCH 피처**와 **자산군별 적응형 앙상블** 구조를 가집니다.

- **주식 카테고리**: R² = 0.51 (시장 효율성으로 랜덤워크에 가까움)

## 7. 입력 변수 정의 (Feature Definitions)

### 7.1 공통 변수

| Variable | Formula | Description | 단위 |
|:---|:---|:---|:---|
| **LogRV** | $\log\left(\frac{252 \times 10000}{22} \sum_{t-21}^{t} r_t^2\right)$ | 22일 로그 실현변동성 | Log scale |
| **Daily Return** | $r_t = \log(P_t / P_{t-1})$ | 일간 로그 수익률 | Decimal |
| **LogRV_lag1** | $\text{LogRV}_{t-1}$ | 전일 변동성 | Log scale |
| **LogRV_lag5** | $\text{LogRV}_{t-5}$ | 5일 전 변동성 | Log scale |
| **LogRV_lag22** | $\text{LogRV}_{t-22}$ | 22일 전 변동성 | Log scale |
| **Target** | $\text{LogRV}_{t+22}$ | 22일 후 변동성 (예측 대상) | Log scale |

### 7.2 V50 전용 시퀀스 구조

| 시점 | LogRV | Return | 설명 |
|:---|:---|:---|:---|
| t-21 | $x_1^{(1)}$ | $x_1^{(2)}$ | 시퀀스 시작 |
| t-20 | $x_2^{(1)}$ | $x_2^{(2)}$ | ... |
| ... | ... | ... | ... |
| t | $x_{22}^{(1)}$ | $x_{22}^{(2)}$ | 시퀀스 끝 (현재) |
| t+1 | - | - | **예측 대상** |
