Universal Model Experiment

통합 모델을 이용한 자산 변동성 예측 (Cross-Asset Volatility Prediction)

1. 통합 모델 개요 (Overview)

본 실험은 23개 주요 자산(Equity, Bond, Commodity)을 하나의 데이터셋으로 통합(Pooling)하여 단일 모델로 학습하는 Universal Model 접근법을 검증했습니다.

핵심 가설 (Hypothesis): 개별 자산마다 별도의 모델을 만드는 것보다, 모든 자산의 데이터를 합쳐서 “변동성의 보편적 법칙(Universal Laws of Volatility)”을 학습하는 것이 노이즈 강건성과 일반화 성능이 더 뛰어날 것이다.

2. 피처 엔지니어링 (Feature Engineering)

설계 원칙 (Design Principles): 1. No Future Info: 시점 \(t\)에서 가용한 정보(\(t-1\)까지의 데이터)만 사용 2. Log-Scale: 모든 변동성 변수를 \(\ln(RV)\)로 변환하여 분포 정규화 및 스케일 통일 3. Cross-Asset: 주식(SPY), 채권(TLT), 금(GLD)의 변동성을 모든 자산 예측에 활용

2.1 피처 구성표 (Feature Config)

그룹 피처 수식 설명
기본 (HAR) LogRV_lag1 \(\ln(RV_{t-1})\) 어제의 변동성
LogRV_lag5 \(\ln(\bar{RV}_{t-5:t-1})\) 주간 평균 변동성
LogRV_lag22 \(\ln(\bar{RV}_{t-22:t-1})\) 월간 평균 변동성
LogRV_Mom \(\ln RV_{t-1} - \ln RV_{t-5}\) 변동성 모멘텀
글로벌 LogVIX \(\ln(VIX_{t-1})\) 내재변동성 (시장 공포)
VIX_Term \(VIX3M - VIX\) 기간 구조 (Term Structure)
SKEW \(SKEW_{Idx} - 1\) 꼬리 위험 (Tail Risk)
교차 자산 LogRV_SPY \(\ln(RV_{SPY})\) 주식 시장 리스크 프록시
LogRV_TLT \(\ln(RV_{TLT})\) 채권 시장 리스크 프록시
LogRV_GLD \(\ln(RV_{GLD})\) 금/원자재 리스크 프록시

2.2 피처 확장 (Feature Expansion) - [신규]

Literature(Feunou 2017, Carr 2009 등)를 기반으로 다음 변수들을 추가하여 모델 개선을 시도했습니다.

그룹 피처 설명 출처/논리
비대칭성 LogRV_Down 하락 변동성 “나쁜” 변동성이 예측력이 높음 (Feunou ’17)
RS_Asymmetry LogRV_Down - LogRV_Up 변동성 비대칭성
VRP VRP_Proxy LogVIX - LogRV 분산 리스크 프리미엄 (Carr ’09)
기술적 지표 RSI 상대강도지수 과매수/과매도 지표
BB_Width 볼린저 밴드 너비 변동성 군집성 측정치
모멘트 Skewness 수익률 왜도 부호화된 점프(Signed Jump) 프록시

3. 실험 결과 (Results)

3.1 아키텍처 비교 (Linear vs Deep Learning)

금융 시계열, 특히 변동성 데이터는 노이즈가 매우 심해 복잡한 딥러닝 모델이 오히려 과적합될 수 있습니다. 본 실험 결과도 이를 뒷받침합니다.

모델 전체 R² 자산 평균 R² 비고
Baseline MLP 0.260 -0.118 기본 비선형 모델
Residual MLP 0.232 -0.134 잔차 연결(Skip-connection) 효과 미미
LSTM -0.046 -0.482 심각한 과적합 발생
Transformer 0.145 -0.218 정형 데이터에 과한 복잡성
Ensemble (최종) 0.848 0.170 선형 + 트리 모델 조합

Model Features R2 Score Improvement
