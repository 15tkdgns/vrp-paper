Multi-Horizon 변동성 예측 모델

V50 모델 개요

V50은 양방향 LSTM과 Temporal Attention을 결합한 변동성 예측 모델입니다. 피처 셀렉션을 통해 선별된 3개 Range 기반 피처(RogersSatchell, GarmanKlass, Range_Close_Ratio)로 중기 예측(60~180d)에서 최고 성능을 달성합니다.

V50 Tuned 핵심 성과 (전방 RV 기준, 11개 자산 풀링)
  • 22d 풀링 R²: 0.780 | 중위 R²: 0.274 (개별 자산 수준 최고)
  • 60d 풀링 R²: 0.796 (전 모델 중 최고)
  • 중기(60~180d) 챔피언: 풀링 R² 및 중위 R² 모두 최고
  • 피처: RogersSatchell_22, GarmanKlass_22, Range_Close_Ratio (3개)

Multi-Horizon 성능 (V50 Tuned, 전방 RV)

성능 해석
  • 풀링 R²는 cross-sectional variation 포함: 자산 간 변동성 수준 차이가 R²에 기여
  • 개별 자산 수준에서는 중위 R² 참조: 22d 기준 0.274로 전 모델 중 최고
  • 1d 예측(R²=0.092)은 모든 모델이 낮음: 단일 일 변동성은 본질적으로 예측 어려움

4. 한계 및 향후 과제

알려진 한계점
  1. Bidirectional LSTM: 시계열 예측에서 역방향 LSTM은 미래 정보를 참조할 수 있다는 학술적 비판이 존재함.
  2. 입력 피처 2개: LogRV + Return만 사용. V71(37개 피처)에 비해 정보량이 부족.
  3. 과적합 방지 미비: 드롭아웃 0, Early Stopping 없이 고정 20 에폭 학습.
  4. IS-OOS Gap: IS(0.848) vs OOS(0.533)로 약 37%의 과적합이 관찰됨.

향후 개선 방향

항목 현재 개선 방향
구조 Bi-LSTM (미래 참조 우려) Unidirectional LSTM 비교 실험
피처 2개 (LogRV, Ret) VIX, Term Structure 등 추가
정규화 없음 Dropout, Early Stopping 적용
검증 Hold-out, Walk-Forward K-Fold 시계열 CV 추가

# V50 모델 개요

V50은 **양방향 LSTM**과 **Temporal Attention**을 결합한 변동성 예측 모델입니다. 피처 셀렉션을 통해 선별된 3개 Range 기반 피처(RogersSatchell, GarmanKlass, Range_Close_Ratio)로 **중기 예측(60~180d)에서 최고 성능**을 달성합니다.

::: {.callout-note}
## V50 Tuned 핵심 성과 (전방 RV 기준, 11개 자산 풀링)
- **22d 풀링 R²**: 0.780 | **중위 R²**: 0.274 (개별 자산 수준 최고)
- **60d 풀링 R²**: 0.796 (전 모델 중 최고)
- **중기(60~180d) 챔피언**: 풀링 R² 및 중위 R² 모두 최고
- **피처**: RogersSatchell_22, GarmanKlass_22, Range_Close_Ratio (3개)
:::

### Multi-Horizon 성능 (V50 Tuned, 전방 RV)

| Horizon | 풀링 R² | 중위 R² | RMSE | 순위 |
|:---|:---|:---|:---|:---|
| 1d | 0.092 | -0.000 | 2.538 | 4위 |
| 5d | 0.577 | 0.101 | 0.793 | 4위 |
| 22d | 0.780 | **0.274** | 0.482 | 2위 (중위 1위) |
| **60d** | **0.796** | -0.178 | **0.446** | **1위** |
| **90d** | **0.788** | **-0.031** | **0.448** | **1위** |
| 120d | 0.751 | **-0.149** | 0.479 | 2위 (중위 1위) |
| **180d** | **0.708** | **-0.328** | **0.511** | **1위** |
| 252d | 0.589 | **-2.199** | 0.590 | 2위 (중위 1위) |

::: {.callout-warning}
## 성능 해석
- **풀링 R²는 cross-sectional variation 포함**: 자산 간 변동성 수준 차이가 R²에 기여
- **개별 자산 수준에서는 중위 R² 참조**: 22d 기준 0.274로 전 모델 중 최고
- 1d 예측(R²=0.092)은 모든 모델이 낮음: 단일 일 변동성은 본질적으로 예측 어려움
:::

## 2. 모델 성능 비교 (1일 예측 모델만)

```{python}
#| echo: false
#| fig-width: 10
#| fig-height: 4

import matplotlib.pyplot as plt
import numpy as np
import json, os

plt.rcParams['font.family'] = 'UnDotum'
plt.rcParams['axes.unicode_minus'] = False

base = '/root/vrp/experiments'
def load_json(path):
    with open(path) as f:
        return json.load(f)

# 1일 예측 관련 모델만 로드
v50 = load_json(f'{base}/creative/v50_results.json')['V50_R2']
v43 = load_json(f'{base}/creative/v43_results.json')['V43_R2']
v14 = load_json(f'{base}/sci/v14_statistical_results.json')
har_ridge = v14['confidence_intervals']['HAR-Ridge']['R2']['mean']
xgb = v14['confidence_intervals']['XGBoost']['R2']['mean']
v50_oos = 0.533 # v51b 공정 비교 결과

models = ['XGBoost\n(V14)', 'V50\n(Hold-out)', 'HAR-Ridge\n(V14)', 'V43\n(Transformer)', 'V50\nOOS (v51b)']
r2_scores = [xgb, v50, har_ridge, v43, v50_oos]
colors = ['#A0AEC0', '#4299E1', '#A0AEC0', '#A0AEC0', '#2B6CB0']

sorted_idx = np.argsort(r2_scores)
models = [models[i] for i in sorted_idx]
r2_scores_sorted = [r2_scores[i] for i in sorted_idx]
colors = [colors[i] for i in sorted_idx]

fig, ax = plt.subplots(figsize=(10, 4))
bars = ax.barh(models, r2_scores_sorted, color=colors, edgecolor='white', height=0.6)

ax.set_xlabel('R² 스코어', fontsize=12)
ax.set_title('1일 예측 모델 성능 비교', fontsize=14, fontweight='bold')
ax.grid(True, alpha=0.3, axis='x')
ax.set_xlim(0.45, 0.60)

for bar, score in zip(bars, r2_scores_sorted):
    ax.text(bar.get_width() + 0.003, bar.get_y() + bar.get_height()/2, 
            f'{score:.3f}', ha='left', va='center', fontsize=10, fontweight='bold')

plt.tight_layout()
plt.show()
```

| 모델 | 출처 | R² | 비고 |
|:---|:---|:---|:---|
| **V50 OOS** | `v51b_results.json` | **0.533** | 공정 비교 (챔피언) |
| V43 (Transformer) | `v43_results.json` | 0.534 | Transformer 아키텍처 |
| HAR-Ridge | `v14_statistical_results.json` | 0.529 | 통계 검증 |
| V50 (Hold-out) | `v50_results.json` | 0.518 | 기본 분할 |
| XGBoost | `v14_statistical_results.json` | 0.502 | 통계 검증 |

### Extreme Market (v59)

```{python}
#| echo: false
#| fig-width: 10
#| fig-height: 4

import json
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'UnDotum'
plt.rcParams['axes.unicode_minus'] = False

with open('/root/vrp/experiments/verification/v59_results.json') as f:
    v59 = json.load(f)

regimes = list(v59.keys())
r2s = [v59[r]['R2'] for r in regimes]

fig, ax = plt.subplots(figsize=(10, 4))
colors = ['#FF6B6B' if r < 0.4 else '#4ECDC4' for r in r2s]
bars = ax.bar(regimes, r2s, color=colors, edgecolor='black')
ax.set_ylabel('R² Score')
ax.set_title('시장 국면별 V50 성능 (OOD 테스트)', fontsize=14, fontweight='bold')
ax.set_ylim(0, 1.0)
ax.grid(True, alpha=0.3, axis='y')

for bar, r2 in zip(bars, r2s):
    ax.text(bar.get_x() + bar.get_width()/2, bar.get_height() + 0.02, f'{r2:.3f}', ha='center', fontweight='bold')

plt.tight_layout()
plt.show()
```

## 4. 한계 및 향후 과제

::: {.callout-warning}
## 알려진 한계점

1. **Bidirectional LSTM**: 시계열 예측에서 역방향 LSTM은 미래 정보를 참조할 수 있다는 학술적 비판이 존재함.
2. **입력 피처 2개**: LogRV + Return만 사용. V71(37개 피처)에 비해 정보량이 부족.
3. **과적합 방지 미비**: 드롭아웃 0, Early Stopping 없이 고정 20 에폭 학습.
4. **IS-OOS Gap**: IS(0.848) vs OOS(0.533)로 약 37%의 과적합이 관찰됨.
:::

### 향후 개선 방향

| 항목 | 현재 | 개선 방향 |
|:---|:---|:---|
| **구조** | Bi-LSTM (미래 참조 우려) | Unidirectional LSTM 비교 실험 |
| **피처** | 2개 (LogRV, Ret) | VIX, Term Structure 등 추가 |
| **정규화** | 없음 | Dropout, Early Stopping 적용 |
| **검증** | Hold-out, Walk-Forward | K-Fold 시계열 CV 추가 |
